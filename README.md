# Metronome

### Simple and beautiful metronome for Android

This project is intended to offer a simple and good-looking metronome for Android.

## Features

* Adjustable beats and subdivisions
* Beat visualization
* Tap tempo
* Light and Dark theme
* No advertisements
* No trackers
* No annoyance
